## 弃用提醒

由于微信版本更新，现极易封号谨慎使用。

## 安装教程

1. 准备：[TRSS-Yunzai](../../../Yunzai)
2. 输入：`#安装WeChat-Plugin`
3. 打开：微信 → 我 → 服务 → 钱包 → 身份信息 → 实名认证
4. 输入：`#微信登录`

## 使用教程

- #微信账号
- #微信登录
- #微信删除 + `账号`

- 注意：
1. 账号离线5分钟，即登录失效，需要重新登录
2. 重新登录后，所有用户 ID 都会改变
